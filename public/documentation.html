About myFlix

myFlix is an API designed to interact with the ... movie database. The end-user will be able to search and retrieve information from the database (e.g. movie titles, actors,directors, genres) as well as to create a personal account and perform a series of other operations (such as updating personal information and creating a list with favourite movies)
